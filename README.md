# Weather App

A simple web app that shows real time weather information for any city in the world.



## Features

- Search weather by city name
- Shows temperature, humidity, and wind speed
- Displays weather icons based on current condition

## Built With

- HTML
- CSS
- JavaScript
- OpenWeatherMap API

## How to Run

No installation needed. Just follow these steps:

1. Clone the repository:
   ```
   git clone https://github.com/maaz-afzal/weather-app.git
   ```

2. Go into the project folder:
   ```
   cd weather-app
   ```

3. Open `index.html` in your browser and you are good to go.

> **Note:** You need a free API key from [OpenWeatherMap](https://openweathermap.org/api). Add your key in the `script.js` file.
